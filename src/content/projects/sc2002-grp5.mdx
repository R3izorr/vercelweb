Group project for NTU SC2002 (OO Design and Programming). The assignment
was to build a CLI management system for Singapore's BTO (Build-To-Order)
housing flow, following SOLID principles in Java.

## Highlights

- Role-based flows for applicant, officer, and manager.
- Persistent state serialized between runs so the graders can reproduce
  scenarios.
- Reports and filters that exercise the OO model instead of hardcoding.
- Javadocs and screenshots included in the repository for the grader's
  convenience.

The repo is preserved here mainly as a record of the work — the code is
intentionally CLI-only and was graded against a specific rubric.
